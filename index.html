Helper Onboarding Process

EzyHelpers - Streamlined Onboarding Workflow

1
Initial Contact
  • Receive lead/reference of helper
  • Call helper to introduce EzyHelpers
  • Explain the onboarding process
2
Collect Personal Details
  • Full Name
  • Gender
  • Age
  • Phone Number
  • Address
  • Native Place
  • Languages Known
  • Experience (Type, Years, Salary)
  • Preferred Job Type & Location
  • Expected Salary
  • Any other details
3
Document Collection
  • Request Aadhar (front & back) via WhatsApp
  • Request recent photo (passport-size/selfie)
4
Aadhar Verification
  • Check document clarity and legibility
  • Verify name matches with provided details
  • Confirm Aadhar number format (12 digits)
  • Validate document authenticity
  • Check for any signs of tampering
  • Ensure both front and back are provided
If Aadhar is invalid:
  • Inform helper about the issue
  • Request new/clearer documents
  • Provide specific feedback
5
Profile Creation
  • Input all collected data
  • Upload photo & Aadhar documents
  • Set status: Verified – Ready
6
Sales Team Notification
  • Add to WhatsApp group/CRM update
  • Wait for matching with customer leads

Ready to Place the Helper?

Once all steps are completed and verified, proceed with placing the helper in a suitable position.
